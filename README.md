# 南宁哈罗礼德学校课后平台
## Harrow LiDe School Nanning Co-Curricular Portal

### 项目概述

这是一个为南宁哈罗礼德学校设计的综合性课后课程规划与奖学金申请平台。该平台旨在帮助学生和家长便捷地规划课后时间，选择精英项目和CCA课程，以及申请各类奖学金。

### 功能模块

#### 1. 主页面 (index.html)
- 学校信息展示
- 核心价值观呈现（勇气、荣誉、谦和、互助）
- 两大功能入口：
  - 课后课程规划
  - 奖学金申请

#### 2. 课后课程规划 (cca-planning.html)
**功能流程：**
- **步骤1：学生信息录入**
  - 学生姓名、年级、学号
  
- **步骤2：精英项目选择**
  - 精英体育校队（足球、篮球、游泳、羽毛球、高尔夫、马术、网球、花样滑冰）
  - 哈罗音乐学院（钢琴、小提琴、吉他、架子鼓、声乐等）
  - 学术竞赛队伍（数学、科学、辩论、模拟联合国）
  - 额外学术支持（英语、数学、科学）
  
- **步骤3：CCA课程选择**
  - 周一至周四 16:00-17:00 时段
  - 自动检测时间冲突
  - 互动式课程卡片选择
  
- **步骤4：确认提交**
  - 选课摘要展示
  - 可视化周课程表
  - 一键提交

#### 3. 奖学金申请 (scholarship.html)
**三个标签页：**

**系统概览：**
- John Lyon Scholarship（全额奖学金）
- Head's Scholarship（半额奖学金）
- Talent Scholarship（优才奖学金）
- 四大评价类别：学术、竞技体育、艺术、领导力与服务

**奖学金规则：**
- 基本申请前提
- 数据与申请机制
- 行为与表现约束
- 评分模型文档下载

**申请入口：**
- 完整的申请表单
- 学生信息
- 类别选择（智能验证）
- 支持材料上传
- 家长信息
- 申请声明

### 设计特色

#### 视觉设计
- **色彩系统：** 英式学院风格，采用深海军蓝、勃艮第红、金色、翡翠绿
- **字体选择：** 
  - 衬线字体：Crimson Pro（优雅、学术感）
  - 无衬线字体：Noto Sans SC（现代、易读）
- **动画效果：** 渐入、滑动、悬停效果，提升用户体验
- **响应式设计：** 完美适配桌面端和移动端（微信访问）

#### 用户体验
- 清晰的进度指示器
- 实时表单验证
- 友好的错误提示
- 流畅的页面过渡
- 直观的视觉反馈

### 技术栈

- **前端：** HTML5, CSS3, JavaScript (ES6+)
- **设计：** 自定义CSS变量系统，模块化样式
- **集成：** 预留飞书多维表格API接口
- **兼容性：** 现代浏览器，移动端优化

### 使用说明

1. **本地预览：**
   - 直接在浏览器中打开 `index.html`
   - 或使用本地服务器（推荐）

2. **部署：**
   - 上传所有文件到Web服务器
   - 确保保持文件结构
   - 配置飞书API密钥

3. **微信集成：**
   - 配置微信公众号菜单
   - 设置跳转链接
   - 测试移动端显示

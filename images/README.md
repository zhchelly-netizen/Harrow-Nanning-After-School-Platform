## 图片处理建议

### harrow-logo.png
1. 裁剪掉多余的白边
2. 确保背景透明
3. 保存为PNG格式
4. 建议分辨率：至少 200x200px

### harrow-hat.png
1. 裁剪掉多余的白边
2. 如果可能，去除背景（保持透明）
3. 保存为PNG格式
4. 建议分辨率：至少 500x400px

### harrow-logo-small-white.png
1. 确保是白色或浅色图案（在深色背景上显示）
2. 背景必须透明
3. 保存为PNG格式
4. 建议分辨率：至少 100x100px

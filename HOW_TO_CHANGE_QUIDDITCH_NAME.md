## 修改步骤

### 方法 1：使用代码编辑器

1. 打开 `harrow-portal/scripts/cca-data.js`
2. 找到第 11-12 行（搜索"机器魁地奇"）
3. 在"机器魁地奇"后面添加"：无人机足球"
4. 保存文件

### 方法 2：使用查找替换

1. 打开 `harrow-portal/scripts/cca-data.js`
2. 使用查找替换功能（Ctrl+H 或 Cmd+H）
3. 查找：`机器魁地奇`
4. 替换为：`机器魁地奇：无人机足球`
5. 点击"全部替换"
6. 保存文件

## 注意事项

### 1. 文本长度
- 新名称比原名称长
- 在移动端可能需要换行显示
- 已有的 CSS 样式会自动处理换行

### 2. 搜索功能
- 如果有搜索功能，确保"无人机足球"也能被搜索到
- 当前系统没有搜索功能，无需担心

### 3. 数据库同步
- 如果将来连接飞书多维表格
- 需要同步更新表格中的课程名称

### 4. 英文翻译
- 如果需要英文版本
- 建议翻译为："Drone Soccer (Quidditch)"

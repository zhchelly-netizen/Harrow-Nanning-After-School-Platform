# 中国新年特效更新 V2

## 更新日期
2026年1月29日

## 更新内容

### 🧧 中国结样式升级

根据传统中国结样式进行了重新设计，采用**盘长结**图案。

#### 新设计特点：

**1. 上部结构**
- 红色挂绳（3px粗细）
- 圆环装饰（直径14px）

**2. 主体盘长结**
- 中心菱形编织图案
- 四个方向的如意云头装饰
- 复杂的编织纹理
- 多层菱形镂空效果
- 中心圆形装饰

**3. 下部流苏**
- 流苏顶部装饰结
- 9条流苏线（中间最长）
- 自然的垂坠感
- 不同长度营造层次感

**4. 颜色方案**
- 主色：#c41e3a（中国红）
- 渐变：#c41e3a → #dc2626 → #991b1b
- 镂空：白色半透明
- 阴影：深色增强立体感

**5. 尺寸**
- SVG viewBox: 100x180
- 实际显示：20-40px（随机）
- 保持原有的大小范围

### ☁️ 云纹装饰升级

改为传统**如意云头**图案，横向排列一排。

#### 新设计特点：

**1. 图案样式**
- 对称的如意云头造型
- 左右各一个螺旋云卷
- 中心装饰圆点
- 内部螺旋纹理装饰

**2. 颜色方案**
- 金色渐变填充（#A69867 → #c4b589 → #d4c49a）
- 深金色描边（#8a7d52）
- 半透明效果（opacity: 0.6-0.9）

**3. 排列方式**
- 横向排列8个云纹
- 使用flexbox均匀分布
- 左右各留5%边距
- 响应式数量调整：
  - 桌面：8个
  - 平板：5个
  - 手机：4个

**4. 动画效果**
- 上下浮动（5px）
- 轻微缩放（1.0 → 1.05）
- 不透明度变化（0.85 → 1.0）
- 每个云纹延迟0.5秒启动
- 4秒循环周期

**5. 位置**
- 位于header底部
- 绝对定位：bottom: -40px
- 高度：80px
- z-index: 20（在header之上）

## 技术实现

### HTML结构
```html
<!-- 中国结容器 -->
<div class="chinese-knots-container" id="chineseKnots"></div>

<!-- 云纹装饰（8个如意云头） -->
<div class="chinese-cloud-divider">
    <svg class="cloud-pattern">...</svg>
    <svg class="cloud-pattern">...</svg>
    <!-- ... 共8个 -->
</div>
```

### CSS关键样式

**中国结**
```css
.chinese-knot {
    position: absolute;
    top: -100px;
    animation: fallDown 10s linear infinite, rotateKnotRight 3s ease-in-out infinite;
    opacity: 0.85;
}
```

**云纹容器**
```css
.chinese-cloud-divider {
    display: flex;
    justify-content: space-around;
    align-items: center;
    padding: 0 5%;
}
```

**单个云纹**
```css
.cloud-pattern {
    width: 60px;
    height: 60px;
    animation: cloudFloat 4s ease-in-out infinite;
}
```

中国结生成函数已更新，使用新的SVG路径绘制盘长结图案。

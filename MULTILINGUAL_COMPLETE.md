## 🎯 使用方法

### 用户端
```
1. 打开任何页面
2. 点击右上角 🌐 按钮
3. 语言立即切换（中文 ↔ English）
4. 刷新页面保持语言选择
```

### 管理端
```
1. 打开首页
2. 滚动到底部右下角
3. 点击小点 · 进入后台
4. 输入密码：Mustslide-0xf6b5
5. 管理所有翻译
6. 点击"保存更改"下载新的 i18n.js
7. 替换到 scripts 目录
```
